	
	1. Создаём тип инфоблока
		При создании типа инфоблока: "Использовать древовидный классификатор элементов по разделам" - выбираем, 
		если есть какая-либо каталогизация элеметнов. Например, категории чего-либо
	2. Создаём инфоблок. Указываем данные для элемента(статьи, слайда и тд.):
		Инфоблок
			Указываем название и символьный код
		Поля
			Указываем картинки (анонс и детальная)
			Текущая дата
			Символьный кол транслитилироваться
		Свойства
			Добляем свои свойства для элемента	
		Поля разделов
			Указывается поля для разделов, например для категории
		Доступ
			Чтение для всех пользователей		
		Подписи
			....
			
	3. Переходим в созданный инфоблок и создаём верхни уровнень (категории)
	4. Выбираем категорию, и в ней создаём элементы
